# ecommerce-sales-dashboard
End-to-end e-commerce sales analysis with data cleaning, EDA, and an interactive Tableau dashboard.

## Project Overview
This project focuses on end-to-end analysis of e-commerce sales data.  
The goal was to clean messy transactional data, perform exploratory data analysis (EDA), and build an interactive Tableau dashboard to uncover business insights.

---

## Tools Used
- **Excel**: Data cleaning and exploratory data analysis
- **Tableau**: Interactive dashboard creation

---

## Data Cleaning & Preparation
Key data preparation steps included:
- Handling inconsistent user IDs (numeric and alphanumeric)
- Fixing mixed and incorrect date formats
- Using final price after discount as the revenue metric
- Cleaning category text values (trimming spaces)
- Removing invalid and incomplete records

---

## Exploratory Data Analysis
The following analyses were performed:
- Monthly revenue trend analysis
- Revenue contribution by product category
- Identification of high-value customers using anonymized user IDs
- Revenue distribution by payment method
---

## Key Insights
- Revenue is concentrated in a small number of product categories
- Monthly sales show variation, indicating seasonality
- A limited group of customers contributes a significant portion of total revenue
- Certain payment methods dominate overall sales

---

## Tableau Dashboard
The interactive dashboard includes:
- Monthly revenue trend
- Revenue by category
- Revenue by payment method
- Top ten buyers
